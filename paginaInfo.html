Sobre el Proyecto

VeciReport es una innovadora aplicación web diseñada para resolver uno de los problemas más comunes en las comunidades costarricenses...

El Problema que Resolvemos

En Costa Rica, muchos problemas comunales permanecen sin resolver durante meses e incluso años...

Nuestra Solución

Una Plataforma Digital Práctica y Cercana

VeciReport permite a cualquier persona reportar un problema con pocos pasos...

Nuestra aplicación web categoriza y visualiza los reportes en tiempo real...

Objetivos del Proyecto

Objetivo General: Desarrollar una aplicación donde los vecinos puedan reportar fácilmente problemas...

  • Definir un sistema de reportes por región...
  • Desarrollar la estructura inicial de la aplicación...
  • Diseñar una interfaz visual atractiva...
  • Implementar funcionalidades dinámicas...

Tecnologías Utilizadas

  • HTML: Estructura del contenido
  • CSS: Estilo y presentación
  • JavaScript: Interactividad
  • PHP: Lógica del servidor
  • MySQL: Base de datos

Funcionalidades Principales

  • Sistema de reportes
  • Gestión de usuarios
  • Panel de administración
  • Sistema de encuestas
  • Políticas de uso
  • Centro de ayuda

Estructura de la Aplicación

  • Inicio: Encuestas y reportes
  • Información: Detalles del proyecto
  • Ayuda: FAQs y contacto
  • Participación: Área de reportes
  • Perfil: Gestión de usuario
